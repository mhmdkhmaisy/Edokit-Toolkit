### Specification 1: 'edokit.ocr.FontSheet' (Complete Metadata Parser)

1. Schema Capabilities:

   - "fontname": String identifier of the font.
   - "basey": Base vertical offset used for character line baseline alignment.

     * "width" / "height": Strict box bounding box parameters for slicing.
     * "advx": Horizontal advance distance (used to jump to the next letter).

### Specification 2: 'edokit.ocr.RasterOCR' (The Scanning Engine)
This class handles text extraction and image preparation loops directly on raw pixel arrays.

1. Method: 'public String readLine(EdokitImage source, int startX, int startY, FontSheet font, int maxColorDistance)'
   - Iterates horizontally across the source frame slice, checking for matching templates in the font's 'glyphMap'.
   - Implement Alt1's specific 'coldif' (color difference) metrics: loop pixel-by-pixel over the candidate glyph's width and height box.
   - For every pixel in the font template that is NOT transparent/empty alpha, compute the absolute RGB variance against the target source screen pixel:
     int deltaR = Math.abs(fontR - srcR);
     int deltaG = Math.abs(fontG - srcG);
     int deltaB = Math.abs(fontB - srcB);
   - A glyph matches if ALL non-transparent pixels fall beneath 'maxColorDistance'.
   - Proportional Spacing: Upon a successful match, advance the cursor coordinate index horizontally by the tracked character's specific 'advX' value. If no glyph matches, advance by 1 pixel to test the next slice.

2. Method: 'public void removeTextPixels(EdokitImage source, int startX, int startY, String verifiedText, FontSheet font)'
   - This implements Alt1's mask technique. Once text (like a yellow countdown timer) is read, this method steps back through the bounding coordinates of that string.
   - For every positive, visible pixel forming the character shapes, overwrite the matching coordinates inside the 'source' byte array to pure black (RGBA = 0, 0, 0, 0).
   - This strips the overlay, preventing the numbers from distorting color sampling checks executed by downstream buff-detection routines.
